# バグ関係
- インストールフォルダがCドライブ以外だと、サムネイルが取得できない

#気になった事
- サムネイルは今一括して同じフォルダに保存されているが、フォルダ事やプロファイル毎のフォルダ下に保存すべきではないか
- 設定を保存する場所を変えられるようになったが、相変わらずプロファイルの場所は固定されているので、プロファイルの設定も変更出来るようにするべき
- ログファイルも開発版とリリース版で同じファイルを共有しているので、そちらも切り分けられているべき、基本的にインストールフォルダに保存すべき

# update.bat の使い方（忘れた時用）
1. 新しいリリースZIPをダウンロードする
2. アプリを置いているフォルダを開く
3. 同じフォルダにある `update.bat` をダブルクリックする
4. ZIP選択ダイアログが出たら、ダウンロードしたZIPを選ぶ
5. 自動で「アプリ終了 → 上書き更新 → 再起動」される

## 注意
- `update.bat` は `MediaArchiver v2.exe` と同じフォルダで実行する
- 更新中はアプリを手動で操作しない
- 失敗したら、もう一度 `update.bat` を実行する（ZIPを選び直す）
