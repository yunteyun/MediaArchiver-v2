型チェックと lint を実行して結果を報告してください。

以下を順番に実行し、エラー・警告を日本語でまとめて報告すること：

1. TypeScript 型チェックを実行する
   ```bash
   npx tsc --build tsconfig.node.json
   ```
   - `tsconfig.node.json` は electron + 共有型のチェック（composite プロジェクト）
   - フロントエンド側（src/）は Vite ビルドで検証するため、build ステップで確認する

2. ESLint を実行する
   ```bash
   npm run lint
   ```

3. 結果を以下の形式で報告する：
   - **型エラー**: ファイル名・行番号・エラー内容を列挙（なければ「なし」）
   - **lint 警告/エラー**: 同様に列挙（なければ「なし」）
   - **総評**: 問題なし / 修正が必要な箇所あり

4. エラーがある場合は修正方針をユーザーに提示し、承認を得てから修正する
