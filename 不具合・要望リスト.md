#発見した不具合（修正完了したらチェックマークを付けること）

##緊急度の高い不具合
- 書庫内の音声ファイル再生時の音量設定が保存されない 
- Lightboxで表示されている動画や画像は原寸大のサイズで表示されているのか、画面からはみ出して表示されている。
また他のUIに被って表示されるため見た目が良くない


##中程度の不具合
- 書庫ファイルの中にフォルダが複数あると、その複数のファイル内にある画像ファイルが表示されない

##軽度の不具合


#要望（ロードマップに追加または重複を確認したらチェックマークを付けること）
- [x] 「外部アプリで開く」の項目で開くアプリを複数設定できるようにする、右クリックメニューに複数のアプリを表示できるようにする。また、設定画面でアプリのパスを設定できるようにする。
- [x] タグ管理の改善。専用のタグ管理画面を追加し、そこでタグの追加、削除、編集、並び替え、カテゴリ分け、色分け、アイコン設定、説明文の設定、タグの検索、タグの自動割り当て（ファイル名やフォルダ名からタグを自動で割り当てる）などを行えるようにする。

##UIUX関係
- [x] 設定アイコンをサイドバーに分かりやすい形で表示する（歯車アイコン＋セッティングの文字など）
- [x] サムネイルの多才なアスペクト比（縦長、横長、正方形など）に対応できるようにする
- [x] 動画ファイルのサムネイルをスクラブ表示の時にカーソルの位置でサムネイルを動かすのではなく、サムネにホバー下段階でサムネイルの自動でパラパラ動かすモードの設定追加。
- [x] スキャン進捗状況表示の自動展開自動格納。横からニョキッと生えるようなアニメーションで表示されると嬉しい。また再度ユーザー自身で表示できるようにして欲しい。サイドバーの境界線の下部にボタンのようなもので出し入れできるようにするとか（ここはAIの意見が聞いてみたい）
- [x] サイドバーのタグリストの展開収納のギミックの追加。カテゴリの横に矢印アイコンなどを追加して、クリックするとタグリストが展開収納されるようにする。また、カテゴリの下に選択したタグを表示し、カテゴリ自体は収納することで今選択しているタグを分かりやすく表示する。
- [x] 音声ファイルの保存された書庫ファイルのサムネイルのアイコンの変更、書庫ファイルかつ音声ファイルである場合は音声ファイル有り別のアイコンを表示するようにする。
- [x] ファイルカードのグループ表示。年、月でグループ化できるようにする。
