## [1.8.0] - 2026-03-10
### Added
- 設定画面に `自動整理` タブを追加し、ルール保存、Dry Run、移動、リネーム、実行履歴、ロールバック確認までをアプリ内で扱えるようにした。
- 自動整理ルールは `手動スキャン後 / 起動時スキャン後 / 監視再スキャン後` に自動実行できるようにし、登録フォルダの整理を継続運用しやすくした。
- 設定画面に `更新` タブを追加し、最新版確認、検証済み ZIP の取得、リリースページ表示、保存先確認をアプリ内から進められるようにした。
- `docs/dev/operations/運用前チェックリスト.md` を追加し、バックアップ、スキャン、更新導線の事前確認項目を整理した（表示への変更なし）。

### Changed
- 設定画面を左ナビ + 右コンテンツの構成へ再設計し、カテゴリ分け、保存スコープ表示、セクション単位の初期値復帰を追加して、設定項目が増えても管理しやすくした。
- バックアップ設定を再起動後も保持するようにし、保存先、保持世代数、自動バックアップ間隔、履歴確認、復元、設定の書き出し / 読み込みを設定画面から継続利用できるようにした。
- プロファイルごとに一覧既定表示、ファイルカード表示、評価表示境界、外部連携の既定値、スキャン / サムネイル設定を持てるようにし、ライブラリごとの運用差を分けやすくした。
- 起動時の自動スキャンは現在のプロファイル設定反映後に始まるようにし、手動スキャン、起動時スキャン、監視再スキャンも互いに干渉しにくい実行管理へ見直した。
- 総合評価の表示と基準を見直し、ファイルカード、左サイドバー、右パネル、中央ビューア、スマートフォルダで同じ色分けと判定基準を使うように統一した。
- 左サイドバーにピン留め、最近開いた場所、開閉式の補助セクション、プロファイルトップを追加し、よく使う場所や統計へ戻りやすくした。
- 設定、統計、右パネル、中央ビューア周辺の読込と描画を見直し、初回表示やスキャン中に UI が重くなりにくいようにした。

### Fixed
- 複数選択したファイルの削除確認が最後の1件だけに置き換わる問題を修正し、選択件数をまとめて確認して削除できるようにした。
- 起動直後の初回スキャンで、プロファイルごとのスキャン条件が反映されないことがある問題を修正した。
- 0.5 刻み評価で最小値や最大値を途中値にした際に範囲外の半星を選べてしまう問題を修正した。
- 重複ファイル結果の縦スクロール、詳細統計タブの warning、ファイル選択時に総合評価バッジが消える問題を修正し、閲覧中の操作安定性を改善した。
- 旧 `★5` タグなどの従来評価タグが統計やタグ一覧へ残り続ける問題を修正し、現在の `総合評価` 軸ベースの表示へ整理した。
