## [1.16.3] - 2026-04-17

### Fixed
- 音声のみの ZIP アーカイブが大量に存在するフォルダを操作しているとき、メインプロセスが 7za コマンドの連続起動でブロックされ、外部アプリ起動の操作が応答しなくなり最終的にアプリがクラッシュする問題を修正した。`getArchiveMetadata` に stat ベースの LRU キャッシュ・in-flight coalesce・p-limit(2) の 3 段並列制御を追加し、同一ファイルへの重複呼び出しと過剰な並列 spawn を防止した。また、画像エントリが存在しないアーカイブの判定結果を 60 秒間キャッシュし、右パネルのプレビューフレーム取得による不要な再呼び出しも抑制した。
