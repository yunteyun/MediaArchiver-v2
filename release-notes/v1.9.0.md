## [1.9.0] - 2026-03-11
### Added
- 設定の `更新` タブに ZIP 同梱のリリースノート表示を追加し、ネットワーク未接続でも更新内容を確認できるようにした。
- 重複チェック画面に `類似候補` モードを追加し、完全一致ではないが名前が同じ・連番違いのファイルも洗い出せるようにした。

### Changed
- 重複チェック画面に一括戦略、保存先フォルダ強調、サイズ / 日時表示の強化を加え、残す対象を判断しやすくした。
- 書庫ファイルカードでホバー時のコマ送り表示に対応し、動画と書庫で設定を分離したうえで、補助表示や画像書庫バッジも追加して一覧で内容を把握しやすくした。
- 登録フォルダ追加時とフォルダ別スキャン設定で、除外する子フォルダを相対パス指定できるようにした。
- 右サイドパネルの動画プレビューに `停止` を追加し、再生 / 停止操作を見出し行へ集約した。

### Fixed
- スマートフォルダ適用中に通常フォルダへ切り替えても絞り込み条件が残る問題を修正した。
- Windows で `\\?\` 付き拡張長パスのターミナルから開発版を起動できない問題を修正した。
- バックアップ復元後の表示消失、スキャン中のプロファイル混入、キャンセル後の進捗崩れなど、運用中に起きやすい安定性問題を修正した。
- 重複チェック画面の判別しづらさと大量件数時の重さを改善し、操作不能になりにくい導線へ整理した。
