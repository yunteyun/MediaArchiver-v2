## [1.11.0] - 2026-03-21
### Added
- 一覧表示に `新着順 / 週ごと / 月ごと / タイプ別` のプリセットを追加し、検索欄の横から見方をすばやく切り替えられるようにした。
- 日付グループ表示に「古い月も週ごとでまとめ続ける」切替を追加し、長い期間を週単位で追いたい使い方にも合わせやすくした。
- 現在の検索や絞り込みをスマートフォルダとして保存できるようにし、よく使う見方を名前付きで再利用しやすくした。

### Changed
- 一覧プリセットの位置や幅を整え、`タイプ` 切替と並べて使っても見やすい上段レイアウトへ整理した。
- 週単位の日付グループ見出しは、同じ月内なら `第n週`、月またぎなら開始日週として表記し、まとまりの意味を読み取りやすくした。
- 検索語、検索対象、タイプ絞り込み、簡易評価フィルタはプロファイルごとに自動保存し、`絞り込み解除` でワンタッチ初期化できるようにした。
- スマートフォルダは並び替え、折りたたみ、編集メニュー、条件アイコン、タグ名まで読める詳細ツールチップに対応し、一覧性と操作性をまとめて改善した。
- 配布ビルド後に Windows で使わない他OS向けバイナリや不要ファイルを削除し、ZIP 配布版を軽くした。
- 用語集や README の案内、旧保管文書の要約を見直し、現行 UI に合った名称と参照先が分かりやすい状態へ整理した。

### Fixed
- フォルダ切替やサイドバー集計で必要なファイルとタグだけを読むようにし、大きいライブラリでも一覧の切替待ちが伸びにくいようにした。
- 大きい登録フォルダのスキャンでは開始前の件数カウントを省略できるようにし、更新監視も変化したサブフォルダ優先で見直すことで待ち時間を抑えた。
- 表示中のカードからサムネイルを先回り生成するようにし、スクロール直後の見た目が整うまでの遅れを軽減した。
- `build:electron` は毎回成果物を作り直してから梱包するようにし、古いアセット名を拾って初回だけ失敗しやすい状態を解消した。
- Node 側の型チェック導線と自動生成ファイルの扱いを整理し、配布や保守で不要なファイルが残りにくい状態へ整えた。
