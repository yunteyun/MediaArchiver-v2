# Lightbox Redesign v2

## 基本方針
- 全画面ライトボックスはやめる
- 情報表示と編集は右サイドバーに寄せる
- メディア表示は中央カラム内だけで完結させる

## レイアウト
- 左: サイドバー
- 中央: 一覧と中央カラム内ビューア
- 右: 情報パネル

## 中央カラム内ビューア
- 表示領域は中央カラムの本文エリアのみ
- ヘッダーや右サイドバーには重ねない
- 背景クリックで閉じる判定も中央カラム内だけに限定する

## 責務
- `CenterViewerRoot`: 表示状態、閉じる、前後移動
- `CenterViewerStage`: 画像 / 動画 / 音声 / 書庫の表示
- `RightPanel`: ファイル情報、タグ、評価、メモの表示と編集

## 操作UI
- `×` ボタンと前後ナビは中央カラム内ビューアの固定UIにする
- 情報エリアをビューア内部に再導入しない

## 追加禁止
- 全画面オーバーレイを復活させない
- 中央カラム外にビューアをはみ出させない
- ビューア内に右パネル相当の情報エリアを重複配置しない

## 後続タスク
- 中央カラム内ビューア整備後は、情報表示と編集導線を右サイドバー側へ寄せる前提で右サイドバーも再整理する
