# Lightbox Redesign v2

## 基本構造
- 左: 情報エリア（固定幅）
- 右: メディアエリア（残り全幅）

## z-index
- 層1: 全画面オーバーレイ背景
- 層2: 左右2エリア
- 層3: `×` ボタン・前後ナビ

## 情報エリア
- 表示項目は `ファイル名` `タグ情報` `評価` `メモ` `ファイル情報` のみ
- 上記以外の常設要素は追加しない

## メディアエリア
- メディアを中央表示する
- 追加ヘッダーや補助表示は置かない

## 操作UI
- `×` ボタンとファイル前後ナビは層3に固定配置する
- 画像サイズや書庫プレビュー配置に追従させない

## 追加禁止
- ワイヤーフレームに無いバッジ・補助パネル・切替UIを追加しない
- レイアウト構造を左右2エリア以外へ拡張しない
