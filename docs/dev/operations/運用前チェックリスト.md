# 運用前チェックリスト

## バックアップ
- バックアップ設定で `自動バックアップ` を有効にする
- 保存先が意図したドライブ/フォルダになっていることを確認する
- `今すぐバックアップを作成` を実行し、履歴に追加されることを確認する
- 最新バックアップのファイルが保存先に存在することを確認する
- テスト用プロファイルで `このバックアップへ復元` を試し、再起動後に復元されることを確認する

## スキャン
- 起動時スキャン ON のフォルダで、起動直後に現在のプロファイル設定どおりにスキャンされることを確認する
- 監視スキャン ON のフォルダで、新規ファイル追加後に自動反映されることを確認する
- 手動スキャン中に別フォルダの更新監視が入っても、進捗表示やキャンセルが破綻しないことを確認する

## プロファイル
- プロファイルを切り替えた直後に、一覧条件と既定表示が切り替わることを確認する
- プロファイルごとのスキャン設定とバックアップ履歴が分離されていることを確認する

## ファイル操作
- 単体削除でゴミ箱経由/完全削除の両方が機能することを確認する
- 複数選択削除で件数表示と実際の削除件数が一致することを確認する
- 移動、リネーム後に一覧表示が即時更新されることを確認する

## 更新
- 更新確認、ZIP ダウンロード、ZIP 適用ダイアログ起動まで一通り実行できることを確認する
- `update.bat` で更新した後、起動と通常操作に問題がないことを確認する
