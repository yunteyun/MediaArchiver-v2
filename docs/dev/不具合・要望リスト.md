## 💡 要望

### 未対応
#### UI関係
- 書庫ファイルでのライトボックスの操作感を改善したが、動画や画像のライトボックスの操作感を同一の物にして欲しい → ROADMAP（全画面ライトボックスをやめ、中央でメディア表示・右で情報表示に役割分担した中央カラム内ビューアへの再設計を継続）

#### システム関係
- ファイル名検索、ファイル名から正規ファイル名を取得（ライトボックスでのファイル名変更は対応済み） → ROADMAP
- 登録フォルダごとのスキャン設定（起動時、起動中ファイル追加された時スキャンするかどうか） → ROADMAP
### 対応済み / ROADMAP反映済み
- FileCard に拡大ボタンを追加し、画像ファイルはホバー時に `360px` 固定の拡大プレビューを表示できるようにした（修正済み, 2026-03-01）
- 右サイドバーにファイルパス、フォルダパス（登録ファイルから下）の表示を追加し、表示順も `ファイル名 / 評価・タグ / メモ / プレビュー / 基本情報` に整理（修正済み, 2026-03-01）
- 右サイドバーの情報量増加に合わせて幅を見直し、動画でも書庫のようにプレビューフレームを確認できるようにした（修正済み, 2026-03-01）
- 右サイドバーの動画フレームからその場面に近い位置で中央ビューアを開けるようにし、上段プレビューも可変高さへ調整して黒帯を抑えた（修正済み, 2026-03-01）
- 右サイドバーの動画プレビュー操作はホバー時のみ表示にし、動画フレームも正方形セルで縦動画を見比べやすくした（修正済み, 2026-03-01）
- 右サイドバー上段プレビューは固定高さへ戻し、背景ぼかしで余白感を和らげて表示位置の揺れを抑えた（修正済み, 2026-03-01）
- 右サイドバー表示中に `標準（M）` と `標準（L）` の大きさ差が潰れる問題を修正し、モードごとのカード幅差が維持されるようにした（修正済み, 2026-03-01）
- `標準（S/簡易）` 以外の表示で右端に余り幅が偏る見え方を調整し、行全体を中央寄せしてバランスを整えた（修正済み, 2026-03-01）
- `標準（M/L）` などはカード幅を少し伸ばせるようにして、右サイドバー表示中でも隙間が空きすぎない密度へ調整した（修正済み, 2026-03-01）
- `標準（M）` は基準サイズを一段小さくして、`標準（S/簡易） / 標準（M） / 標準（L）` が 1 列ずつ増減しやすい密度へ調整した（修正済み, 2026-03-01）
- `標準（L）` は基準サイズを少し大きめに戻し、`標準（M）` より一段大きい表示として見分けやすくした（修正済み, 2026-03-01）
- `動画（ワイド）` も基準サイズを少し大きくして、右サイドバー表示中でも横方向の空白が目立ちにくい密度へ調整した（修正済み, 2026-03-01）
- 漫画モードも基準サイズを少し大きくして、縦長カードとしての見やすさを保ちつつ空白が目立ちにくい密度へ調整した（修正済み, 2026-03-01）
- 中央カラム内ビューアの書庫表示に単独表示と前後送り、グリッドへ戻る導線を追加し、書庫内の確認を中央ビューア内だけで完結できるようにした（修正済み, 2026-03-01）
- 中央カラム内ビューアの書庫グリッドは正方形サムネイルへ揃えて少し大きめに見えるよう調整し、ページ一覧を見比べやすくした（修正済み, 2026-03-01）
- 中央カラム内ビューアの書庫グリッドは `6枚 / 正方形` の一覧表示へ戻しつつ、各サムネイルを `contain` ベースで見切れにくくした（修正済み, 2026-03-01）
- 中央カラム内ビューアの書庫プレビューフレームは黒帯を残しつつ背景色を少し柔らかくし、真っ黒さが浮きにくい見え方へ調整した（修正済み, 2026-03-01）
- 起動時を含むスキャン進捗ウィンドウは完了後に自動で畳み、左サイドバーから結果を任意に再表示できるようにした（修正済み, 2026-03-01）
- スキャン結果を左サイドバーから再表示した後は自動で閉じないようにし、確認用に開き直した時の保持を改善した（修正済み, 2026-03-01）
- スキャン進捗ウィンドウの完了/エラー時ボタンは最小化に統一し、結果の破棄は左サイドバー側だけで行うよう整理した（修正済み, 2026-03-01）
- GIFなどのアニメーション画像のサムネイルマウスオーバーでアニメーションさせる（修正済み, 2026-02-25）
- pngのアニメーション画像（apng）への対応（修正済み, 2026-02-25）
- 左サイドバーのフォルダ表示で登録したフォルダ下に子フォルダ孫フォルダがある場合でもツリー表示出来るようにする（修正済み, 2026-02-26）
- ライトボックス情報パネルの評価/タグ編集UIを共通操作へ統一し、ファイル名クリック変更と詳細情報表示を強化（修正済み, 2026-02-27）
